Datenschutzerklärung – Barricade

Letzte Aktualisierung: 2. März 2026

Barricade nimmt Ihre Privatsphäre sehr ernst. Diese Richtlinie erklärt, welche Informationen erfasst werden, wie sie verwendet werden und welche Rechte Sie als Nutzer haben.

1. Informationen, die wir nicht erfassen

Barricade erfasst, speichert oder teilt keine personenbezogenen Daten wie Ihren Namen, Ihre E-Mail-Adresse, Ihren Standort oder Ihren Browserverlauf.

2. Barrierefreiheit

Um die Funktionen zum Blockieren ausgewählter Apps und Websites bereitzustellen, benötigt Barricade Zugriff auf den Barrierefreiheitsdienst Ihres Geräts.

  • Diese Berechtigung wird ausschließlich verwendet, um zu erkennen, welche Apps und Websites genutzt werden.
  • Es werden keine Informationen an externe Server gesendet. Alles wird lokal auf Ihrem Gerät verarbeitet.

3. Kamera

Um die Funktionen zum Scannen von QR-Codes und Barcodes bereitzustellen, benötigt Barricade Zugriff auf die Kamera Ihres Geräts.

  • Diese Berechtigung wird ausschließlich zum Scannen von QR-Codes und Barcodes beim Entsperren einer App oder Website verwendet.
  • Es werden keine Informationen an externe Server gesendet. Alles wird lokal auf Ihrem Gerät verarbeitet.

4. Rechte der Nutzer

  • Recht auf Information: Sie können erfahren, was die App mit Informationen macht, auch wenn diese minimal sind.
  • Recht auf Transparenz: Sie können diese Datenschutzerklärung jederzeit einsehen.
  • Recht auf Widerruf von Berechtigungen: Sie können die Berechtigung für den Barrierefreiheitsdienst jederzeit in den Geräteeinstellungen widerrufen, nachdem Sie die von Ihnen konfigurierten Codes gescannt haben.
  • Recht auf Kontaktaufnahme mit dem Entwickler: Bei Fragen oder Anliegen zum Datenschutz können Sie eine E-Mail an die unten angegebene Adresse senden.

5. Änderungen der Richtlinie

Wir behalten uns das Recht vor, diese Richtlinie zu aktualisieren. Änderungen werden in diesem Abschnitt veröffentlicht und – sofern relevant – innerhalb der App mitgeteilt.
